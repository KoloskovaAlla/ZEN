1. Авторизация в firebase
2. Установка CLI (command line interface) firebase-tools
   npm i -g firebase-tools
3. Проверить, установлен ли firebase:
   firebase --version 
4. Залогиниться:
   firebase login
5. Проверить, есть ли доступ ко всем проектам
  firebase projects:list
6. В случае ошибки - выйти из firebase и войти
   firebase logout
   firebase login
7. Инициализация приложения (если еще нет текущего проекта в списке проектов в firebase)
  firebase init 
8. Шаг настройки проекта. Выбрать существующий проект. (Выбор ответа: перемещение по вариантам ответа - стрелками "вверх" и "вниз", выбор ответа - клавиша пробел)
9. Выбрать нужную опцию - хостинг
   Указать папку - build или переименовать папку public в build
10. Настроить как одностраничное приложение? - Y(es) 
11. Настроить автоматические сборки и развертнывания с github - N(o)
12. Нужно ли перезаписать build/index.html? N(o)
13. Инициализация firebase завершена
14. Выполнить deploy:
    firebase deploy
